# vue cil 3 模仿去哪儿 移动端 web

## 技术栈和主要框架
📦 Vue 全家桶：vue2.5 + vuex + vue-router + webpack

📌 ES6

📡 网络请求：axios


### 项目运行
```
git clone https://github.com/LiMiu331/Travel.git

cd Travel

npm install 或 yarn(推荐)

npm run serve
```

### 关于接口数据
```
此项目的所有接口数据来源于本地前端mock文件夹下的数据。
```
